# KuroTool
### 使用方法:
1、添加Jcenter仓库
```
buildscript {
    repositories {
        ...
        jcenter()
        ...
    }
}
```

**Gradle依赖：**
```kotlin
implementation 'com.trs.kuroTool:kuroTool:1.0.0'
```
或者

**Maven**
```
<dependency>
	<groupId>com.trs.kuroTool</groupId>
	<artifactId>kuroTool</artifactId>
	<version>1.0.0</version>
	<type>pom</type>
</dependency>
```

之后将长期维护
